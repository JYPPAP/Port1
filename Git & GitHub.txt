GitHub에 코드를 올리는 과정
1. 내 컴퓨터에서 프로젝트 폴더에 여기서 Git을 쓴다고 명령 
	git init
2. 코딩
3. 내가 변경한 파일 중 올리길 원하는 것만 선택
	git add
4. 선택한 파일을 한 덩어리로 만들고 설명 적기.
	git commit -m "첫페이지 제작"
5. GitHub 사잍트에 프로젝트 저장소 만들기
6. 내 컴퓨터 프로젝트 폴더에 GitHub 저장소 주소 알려주기
	git remote add
7. 내 컴퓨터에 만들었던 덩어리를 GitHub에 올리기
	git push
